# 烟花模拟器

一个可以模拟各种烟花效果的网页， 它可以展示各种绚丽多彩的烟花效果 并让人仿佛置身于真实烟花的绚丽世界之中

</div>

## 功能特性

### 🎆 丰富的烟花效果
- **多种烟花类型**：支持25种不同的烟花效果，包括菊花、圆环、螺旋、彩虹等
- **真实物理模拟**：基于物理引擎的烟花爆炸和粒子效果
- **音效支持**：逼真的烟花爆炸和升空音效

### 🌍 多语言支持
- **中英文切换**：完整支持中文和英文界面
- **智能语言检测**：自动根据浏览器语言设置默认语言
- **本地存储**：记住用户的语言选择偏好
- **完整翻译**：包括界面文本、菜单选项、帮助内容等

### ⚙️ 自定义设置
- **烟花大小调节**：支持3"到16"不同尺寸的烟花
- **画质调节**：低、正常、高三种画质模式
- **天空照明效果**：可调节烟花爆炸时的背景照明强度
- **缩放控制**：50%到200%的视角缩放

### 🎯 交互体验
- **自动发射模式**：自动连续发射烟花
- **手动点击发射**：点击屏幕任意位置发射烟花
- **文字烟花**：烟花形成文字效果（如"新年快乐"等）
- **隐藏控制按钮**：提供沉浸式观赏体验

## 多语言翻译功能

### 支持的语言
- **中文（简体）**：默认语言
- **英文（美国）**：国际用户友好

### 翻译范围
- **界面文本**：页面标题、按钮标签、菜单标题
- **选择器选项**：烟花类型、画质设置、天空照明等
- **帮助内容**：详细的设置说明和功能介绍
- **加载界面**：加载状态和提示信息

### 使用方法
1. 打开设置菜单
2. 在"语言"选项中选择需要的语言
3. 界面将立即切换至所选语言
